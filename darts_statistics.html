Darts statistics¶

This notebook serves as a practice of the fundamentals of data analytics. It uses a database containing quasi-random data generated by a python script that consider "highscore" practice sessions of 33 rounds, aiming at 20 and 19, with pre-defined accuracy and probabilities. Therefore the focus of this notebook is rather on handling data and visualisation, than the analysis itself.

The key skills:

  • using pandas dataframes (creating, modifying, grouping, merging and converting columns, etc.)
  • visualisation techniques and formatting wiht matplotlib and seaborn
    • bar chart
    • line plot
    • pie chart
    • histogram
  • handling time series (converting to datetime, using them as index, resampling, etc. )
  • determining key metrics

1.1. Database format:¶

  • round_nr: number of the ongoing round, one round consits of 33 throws (visits)
  • throw_nr: number of the throw (1-33)
  • dart_nr: number of the thrown dart (1-3)
  • prefix: field identifier: S - simple, D - double, T - treble
  • score: sector that was hit
  • value: thrown score (T20 -> 60)

Monthly Pie Charts showing the top 5 most hit score in 2022

# Setting up dataframe with merged score column (ex: S, 20 -> S20)
df_field = df.copy()
df_field["field"] = df["prefix"] + df["score"].astype(str)

# Creating main plot layout with title
fig, axs = plt.subplots(4, 3, figsize=(10,12), facecolor='white')
fig.suptitle("Top 5 Scores in 2022", fontsize=20, y=1.01)
plt.tight_layout()

# Defining constants for plots
TOP_N = 5
colors = [ np.append(plt.get_cmap(color_name)(np.linspace(0.6, 0.2, TOP_N)), [[0.7, 0.7, 0.7, 1]], axis=0) 
          for color_name in ["Reds", "Blues", "Greens"] ]
          
# Looping through months
month_start = pd.to_datetime("2022-01-01")
for i in range(12):

    # Filtering dataframe to the specific month
    month = month_start + pd.DateOffset(months=i)
    df_field_month = df_field[(df_field.index > month) & 
                            (df_field.index < month + pd.DateOffset(months=1))]

    # The top 5 scores 
    pie_top5 = df_field_month["field"].value_counts(sort=True, ascending=False)[:TOP_N]
    
    # Adding all the others as a cummulated pie slice
    count_others = df_field_month["field"].value_counts(sort=True, ascending=False)[TOP_N:].sum()
    pie_top5 = pd.concat([pie_top5, pd.Series(data={"Other":count_others})])

    # Creating and formatting the pie plot    
    row, col = i // 3, i % 3
    axs[row, col].pie(
        pie_top5,
        labels=pie_top5.index,
        colors=colors[i % 3],
        autopct='%1.1f%%',
        pctdistance=0.7,
        startangle=180,
        counterclock=True,
        wedgeprops={"linewidth": 1, "edgecolor": "white", "width": 0.6},
        textprops={'size': 'x-small'},
        frame=True
        )
    
    # Removing ticks and labels to have frame only
    axs[row, col].set_xlim(-1.5, 1.5)
    axs[row, col].set_xticklabels([])
    axs[row, col].set_xticks([])
    axs[row, col].set_ylim(-1.3, 1.3)
    axs[row, col].set_yticklabels([])
    axs[row, col].set_yticks([])

    # ADding title
    axs[row, col].set_title(month.month_name())

4.3. Longest streak of 20s¶

Finding the longest streak for the sector 20. All fields count (simple, double, treble). A streak is broken if another sector is hit, or the practice ends for the day.

In [ ]:
rows = df.shape[0]
streaks = []
streak, max_streak, start = 0, 0, 0

for r in range(rows):
    
    # Checking if sector 20 is hit
    if df.iloc[r,4] == 20:
        # Checking if the practice did not end for that day (day in the index stays the same)
        if df.index[r] == df.index[r-1]:
            streak += 1
        else:
            streak = 1
            if streak > max_streak:
                max_streak = streak
                start = r - max_streak

    else:
        if streak > max_streak:
            max_streak = streak
            start = r - max_streak
        streak = 0        
    streaks.append(streak)

print("Longest streak of 20's: ", max_streak)
